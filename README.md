Linkbot
=======

Here's a short demo of blather with a tiny sinatra app.

The Usual
=========

* Install the gems

        bundle install

* Start the server and blather client

        foreman start

* Go to [http://localhost:9292/](http://localhost:9292/)

* Send a message to developer@highgroove.com with this format:

        Add Link: <url>

=========
